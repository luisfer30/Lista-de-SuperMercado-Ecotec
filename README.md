# Lista-de-SuperMercado-Ecotec
Lista para Tarea de Ecotec
## Descripción del proyecto
Este proyecto consiste en una aplicación web tipo **lista de supermercado**, desarrollada como parte de la asignatura **Programación de Sistemas Web – ECOTEC**.

La aplicación permite al usuario agregar productos con su cantidad, marcarlos como comprados, eliminarlos y mantener la información almacenada aunque se recargue la página, utilizando tecnologías web básicas y buenas prácticas de desarrollo.

## Funcionalidades implementadas
- Agregar productos dinámicamente sin recargar la página.
- Validación de campos obligatorios (producto y cantidad).
- Validación de cantidad mayor a cero.
- Mensajes interactivos usando **SweetAlert2**.
- Visualización de productos en una **tabla HTML**.
- Marcar productos como **comprados / pendientes**.
- Eliminación de productos de la lista.
- Contadores dinámicos:
  - Totauctl de prodos
  - Productos comprados
  - Productos pendientes
- Persistencia de datos mediante **localStorage**.
- Recuperación automática de la lista al recargar la página.

## Tecnologías utilizadas
- **HTML5** (estructura semántica)
- **CSS3** (estilos, diseño y estados visuales)
- **JavaScript** (DOM, eventos, createElement, classList)
- **SweetAlert2** (mensajes y alertas)
- **GitHub** (control de versiones)
